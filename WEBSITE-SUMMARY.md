## 🔧 Customization Guide

### Change Main Color
```css
/* In files/style.css, line 7 */
--main-color: #YOUR_COLOR;
```

### Add New Animation
```javascript
// In files/script.js
anime({
    targets: '.your-element',
    translateY: [0, -20],
    opacity: [0, 1],
    duration: 800,
    easing: 'easeOutExpo'
});
```

### Adjust Parallax Speed
```javascript
// In files/script.js, line 35
element.style.transform = `translateY(${scrolled * 0.5}px)`;
// Change 0.5 to adjust speed
```
